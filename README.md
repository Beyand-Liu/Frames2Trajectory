# Frames2Trajectory
This is a code that converts videos into motion trajectories. It combines an image sequence into one image to display the objects's trajectory and oriention in it. Please refer to the following Bilibili link for the detailed explanation of how the code runs.The folder "JiayinHeJin1" contains sample image sequences. <br />
https://www.bilibili.com/video/BV1vx4y1U7xd/. <br />
